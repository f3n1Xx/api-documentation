SQLSerializer

public protocol SQLSerializer

Capable of serializing SQL queries into strings. This protocol has free implementations for most of the requirements and tries to conform to general (flavor-agnostic) SQL.

You are expected to implement only the methods that require different serialization logic for your given SQL flavor.
